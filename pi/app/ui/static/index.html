The current scene name shows what's playing on the LEDs right now. Presets are saved combinations of effect + parameters — tap to recall them.

Use the brightness slider above to control overall LED brightness. "Blackout" turns all LEDs off immediately; "Resume" restores output.

The auto-brightness checkbox enables sunrise/sunset automation (requires location config).

Preview animations without changing the live LEDs.

Select an effect from the dropdown and tap "Preview". The pixel grid shows what the animation looks like — each dot represents one LED matching your physical strip arrangement.

Use "Fit" to scale the canvas to fit the screen, or "1:1" for actual pixel size (scrollable). Preview runs independently from the live output.

Tap "Stop" to end the preview. The preview disconnects automatically when you leave this tab.
